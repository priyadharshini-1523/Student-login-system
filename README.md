Student Login System

📌 Description
This project is a basic student login system developed using HTML, CSS, and JavaScript.
It demonstrates how user details can be added and validated in a simple web application.

🚀 Features
- Student login form  
- Add and validate user details  
- Basic input validation  
- Simple and user-friendly interface  

🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
